********* Cách thực thi chương trình *********
1/ Chạy .\Cài đặt\QLHS.msi để tiến hành cài đặt
2/ Click vào biểu tượng QLHS trên desktop
3/ Chương trình sẽ khởi động form "Kết nối cơ sở dữ liệu" cho chương trình:
  + Gõ tên server
  + Nhấn Connection
  + Chọn database
  + Nhấn OK
=> Chương trình sẽ khởi động lại và đã kết nối cơ sở dữ liệu thành công
4/ Chọn 1 trong các user sau đăng nhập vào chương trình:
